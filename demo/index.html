jDropdown


A JQuery plugin for quickly create drop-down controls in bootstrap style
快速生成Bootstrap风格下拉框的JQuery插件

Quick start

include the files

    <script src="./js/jquery.min.js"></script>
    <script src="./js/bootstrap.min.js"></script>
    <script src="./js/jdropdown.min.js"></script>

call the the plugin on a element as your wish

    $("#element").jDropdown(method, options);

here 's an example for how to initizlize and load data for a drop-down control

    $(document).ready(function () {
        $('#drop-ctrl').jDropdown('init', {
            width: '200px'
        });
        var dataList = [
            { key: '1', value: 'January' },
            { key: '2', value: 'February' },
            { key: '3', value: 'March' },
            { key: '4', value: 'April' },
            { key: '5', value: 'May' },
            { key: '6', value: 'June' },
            { key: '7', value: 'July' },
            { key: '8', value: 'August' },
            { key: '9', value: 'September' },
            { key: '10', value: 'October' },
            { key: '11', value: 'November' },
            { key: '12', value: 'December' }
        ];
        $('#drop-ctrl').jDropdown('loadData', dataList, {
            key: 'key',
            value: 'value',
            defaultValue: 'Select',
            func: function (msg) {
                alert('You selected "' + msg + '"');
            },
            separator: [3, 6, 9]
        });
    });



All done!

Options

you can custom the control by passing options as second parameter when calling method 'init', 'load', 'update' all available options are listed below.

Option type default Description
isSplit bool true Whether the title of drop-down display in a single element on the left of the drop-down button.
titleStyle string 'span' Witch will be use for title label, span or button
width string 'auto' The width of the title label. e.g. 200px , 50%
isUp bool false Popup the menu by specify this attribute 'true'
defaultValue string null Default key of drop-down menu, which will be displayed on the title label.
separator array undefined Specify position of the separators by int array.
menuAlign string 'left' Specify alignment of drop-down menu 'left' or 'right', the attribute 'width' should be 'auto'.
disabledKeys array undefined Specify disabled items of drop-down menu according to key
func function undefined The function will be called when item of the drop-down menu is clicked, whose key will be passed to the function.

Methods

Specify method name as first parameter when calling jDropdown on the controls.

init

init drop-down control with settings

    $('#drop-ctrl').jDropdown('init', {
        isSplit: true
    });

load

load drop-down control with data list and settings

    var dataList = [
        { key: '1', value: 'January' },
        { key: '2', value: 'February' },
        { key: '3', value: 'March' },
        { key: '4', value: 'April' },
        { key: '5', value: 'May' },
        { key: '6', value: 'June' },
        { key: '7', value: 'July' },
        { key: '8', value: 'August' },
        { key: '9', value: 'September' },
        { key: '10', value: 'October' },
        { key: '11', value: 'November' },
        { key: '12', value: 'December' }
    ];
    $('#drop-ctrl').jDropdown('loadData', dataList, {
        key: 'key',
        value: 'value',
        defaultValue: 'Select',
        func: function (msg) {
            alert('You selected "' + msg + '"');
        },
        separator: [3, 6, 9]
    });

update

update settings for drop-down control

    $('#drop-ctrl').jDropdown('update', {
        width: '500px',
        separator: [4, 8],
        titleStyle: 'span'
    });

insertItem

insert item to data list of drop-down control

    $('#drop-ctrl').jDropdown('insertItem', [
            { key: '13', value: 'January' },
            { key: '14', value: 'February' }], {
        isUp: true
    });

clickItem

tigger click event of the specify item of drop-down control

    $('#drop-ctrl').jDropdown('2');

removeItem

remove item from data list of drop-down control

    $('#drop-ctrl').jDropdown('removeItem', [2, 5, 7], {
        isUp: false
    });

Compatibility

Requirs jQuery 1.7+ & bootstrap 3.0.3+

Works well with all modern browsers and IE8+.
